# password_generator

A simple password generator written in Python.

Repository: https://github.com/JunayedMh/password_generator

## Overview

This project generates random passwords using letters, numbers, and symbols.

## Requirements

- Python 3 installed on your system

## Usage

1. Open a terminal or command prompt.
2. Change to the repository folder:
   ```
   cd password_generator
   ```
3. Run the generator:
   ```
   python pass_generator.py
   ```

## Notes

- Make sure Python 3 is available in your PATH.
- The password output appears in the terminal.
- The required length of the password is minimum 12.
